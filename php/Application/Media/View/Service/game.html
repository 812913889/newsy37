手机游戏无法登录怎么办？

1、版本过低，游戏常会更新，注意版本。2、数据错误：手机上的数据损坏，导致游戏崩溃。3、手机运行内存不足。4、网络连接错误。5、手机配置过低。6、手机时间与网络时间不同步。7、游戏更新维护．如果还是无法解决，玩家可以联系我们的在线客服咨询哦。客服QQ：800089273．


充值成功未到账的原因？

　　由于网络因素、频繁操作等方面原因，提示成功的充值可能不一定会立即到达游戏内，如果超过10分钟还没有到账，就可能是某些环节出错了。

　　可能出现这两种情形：充值并没有成功划拨、充值成功划拨了金额但是没有到游戏，所以如果小伙伴碰到此类问题，请先查询一下具体是哪类问题哦。

　　对于充值并没有成功划拨的情况，通常发生在各类充值卡的充值方式下。

　　我们所有的游戏都是采用易宝支付渠道（包括移动充值卡，联通充值卡、盛大卡），由于网络等不稳定因素可能会存在充值不能成功划拨的情况。


安卓手机如何清理缓存？

　　1、进入手机设定（设置）-选择应用程序管理器（应用）

　　2、然后找到我们的游戏；

　　3、点击清除数据，就可以清理得干干净净了。



手机如何截图？

　　1.HOME (首页键) + 电源键

　　此方式适用有实体按键的装置 (Note、S2、S3)，跟iOS的截图方式一样，按下HOME+电源键就可截图。

　　2.Volume Dowm (音量 - 下键) + 电源键

　　无实体按键装置基本上会是用这截图方式，HTC装置大多适合使用此种方法，不过依照装置会有"先按音量再按电源" 或 "同时按下"的差别，有实体按键的机种也会因为版本型号的不同而有可能需使用此种方式截图。
